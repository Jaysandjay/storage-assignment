API stands for Application Programming Interface. They are predefined functions which allows different software components interact with each other. In this Assignment, I am using the Web Storage API. This allows me to let the user save data on their browser. Once Data is stored Using the Web API, the user can refresh the page and still have their previous information or settings still saved.

Other Commonly used HTML5 APIs are the Geolocation API and the Canvas API. The Geolocation API allows the web page to access the user's current geographical position in latitude and longitude. The Canvas API provides a 2D drawing surface for the user which gives the user helpful tools to create animation, games and drawings
